# Monte Carlo Portfolio Simulator

A Streamlit web application for Monte Carlo portfolio simulations, designed for European investors. The app is in Italian.

## Features

- Simulate portfolio performance using Monte Carlo methods
- Fetch real-time stock data using Yahoo Finance
- Visualize simulation results with charts
- User-friendly interface in Italian

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mrazzini/montecarlo_streamlit.git
   cd montecarlo_streamlit
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the Streamlit app:
```bash
streamlit run app.py
```

Open your browser to the provided URL and interact with the simulator.

## Deployment

Deploy to Streamlit Cloud:
1. Push your code to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub repo and deploy

## Requirements

- Python 3.7+
- streamlit
- yfinance
- numpy
- pandas
- matplotlib
